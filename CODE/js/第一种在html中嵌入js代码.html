javascript是一门脚本语言,什么是脚本语言

所有脚本语言都有这样的一个特点;脚本语言最终运行的哪个程序是以普通文本的形式保存的,java就不是脚本于洋,java语言需要先编写xxx.java源文件，在源文件中编写源代码，然后Java源代码还需要进行一个编译，编译之后会生成xxx.class文件,而最终运行的class文件,class文件，class文件不是普通文本，用记事本打不开,所有java不是脚本语言,javascript是脚本语言

html css javascript他们的关系

html 是主体 css 是用来修饰html的 js 是用来操作html和css的，修改html中的元素，修改css样式，达到网页的动态效果。
  • 需求:在页面中有一个按钮，用户点击这个按钮的时候弹出对话框
  • 2.javascript是一种基于“事件驱动”的编程语言，一般就是依靠事件来触发js代码执行的 例如：鼠标单击事件click。鼠标双击事件: dblcick,获取焦点事件;focus失去焦点事件 blur
  • 在js当中任何一个事件都有对应的一个“事件句柄”，所有的事件名称前面添加 “on” click 对应的事件句柄: onclick 单击事件 dblcick 对应的事件句柄: ondblcick 双击事件 foucs 对应的事件句柄: onfouce 获取焦点事件 blur 对应的事件句柄: onblur 失去焦点事件
  • 注意：所有的事件句柄都是以 "标签属性存在的"
  • onclick 后面的js代码什么时候执行呢 不会再页面打开的时候执行，只有当鼠标单击blick事件发生之后，才会执行这个js代码
  • 再js中有一个内置的隐含对象: window:注意是全部小写的,window:代表的是浏览器窗口对象，可以直接拿来使用,window对象有一个alert方法/函数，他可以完成弹窗操作
